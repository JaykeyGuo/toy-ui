### 车牌号组件

```vue
<template>
  <car-input
    v-model="carNo"
    :showDisplay="true"
  />
</template>
```

通过 `showDisplay` 属性来控制是否展示预览车牌号。
本组件可兼容新能源车牌和部分特种车牌，通过车牌号的类型对输出事件有严格控制。
